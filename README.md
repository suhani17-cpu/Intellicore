# Intellicore - AI Study Assistant

## About
Intellicore is an AI study assistant that explains programming 
concepts through simple examples for engineering students.
It provides instant solutions to technical questions.

## Features
- Ask programming or engineering-related queries
- Understand answers with illustrative examples
- Keeps track of chat history
- User-friendly web UI

## Technologies Used
- Programming Language: Python
- Web Framework: Streamlit
- AI Model: Groq (llama-3.3-70b-versatile)

## Deployment Steps
1. Install dependencies:
   pip install streamlit groq
2. Put your API key inside app.py
3. Execute the program:
   streamlit run app.py
